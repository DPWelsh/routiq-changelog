This is where we’ll post what’s new at Routiq — features, fixes, the lot. We’re keeping it short and unembellished. If we shipped something, it lands here.

Two sections worth knowing about:

- **[Changelog](/)** — the running list of what went out. Sorted newest-first. Subscribe via [RSS](/feed.xml).
- **[Roadmap](/roadmap)** — what’s next, what’s on deck, what’s parked. We update this as plans evolve and won’t put dates on it because deadlines on public roadmaps age badly.

If something’s missing or wrong, the repo is at [github.com/DPWelsh/routiq-changelog](https://github.com/DPWelsh/routiq-changelog) — PRs welcome.
